# 🐾 Pata Certa

O **Pata Certa** é um sistema web desenvolvido para conectar animais em busca de um lar a potenciais adotantes. O projeto foca na facilitação do processo de adoção e na gestão de ONGs parceiras, proporcionando uma interface intuitiva para a visualização de pets e cadastros.

## 🚀 Tecnologias Utilizadas

Este projeto utiliza uma stack baseada em Python para o backend e tecnologias web padrão para o frontend:

*   **Backend:** Flask (Python)
*   **Frontend:** HTML5, CSS3 (Globals.css) e JavaScript
*   **Arquitetura:** Padrão de rotas organizadas em Views e Templates

## 📂 Estrutura do Projeto

De acordo com a organização atual do repositório:

- `static/`: Contém arquivos de estilo (CSS), scripts (JS) e imagens dos produtos e interface.
- `templates/`: Arquivos HTML responsáveis pela interface do usuário (Adoção, Cadastro de ONGs, Exibição de Pets).
- `views/`: Lógica de rotas separada entre API e páginas principais (`main.py`).
- `wsgi.py`: Ponto de entrada da aplicação.
